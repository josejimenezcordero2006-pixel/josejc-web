Rack de telecomunicaciones

Este proyecto muestra el montaje completo del rack de telecomunicaciones de la oficina, incluyendo el switch PoE para cámaras, el router principal, el servidor y la organización del cableado estructurado.

Detalles del proyecto

  • Rack mural con ventilación superior.
  • Switch PoE para alimentar cámaras IP con mas puertos para futuras ampliaciones.
  • Router principal con salida a Internet y VLANs configuradas.
  • Servidor para copias de seguridad y gestión interna.
  • Organización del cableado con guías y bridas de velcro.
  • Patch panel para distribución ordenada de los puestos de trabajo.
